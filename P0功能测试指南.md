# P0 功能测试指南

## 🎯 测试目标

## ✅ 测试清单

### 1. Toast 通知系统

#### 1.1 任务添加通知
- [ ] 打开任务列表（点击左下角任务图标）
- [ ] 在输入框中输入任务名称
- [ ] 按 Enter 或点击外部添加任务
- [ ] **预期结果**：右上角显示绿色"任务已添加"提示，3秒后自动消失

#### 1.2 任务完成通知
- [ ] 点击任务左侧的圆圈，标记为完成
- [ ] **预期结果**：显示绿色"任务已完成 ✓"提示

#### 1.3 任务取消完成通知
- [ ] 点击已完成任务的对勾图标
- [ ] **预期结果**：显示蓝色"任务已标记为未完成"提示

#### 1.4 任务删除通知
- [ ] 悬停在任务上，点击右侧的 ✕ 按钮
- [ ] 在确认对话框中点击"删除"
- [ ] **预期结果**：显示蓝色"任务已删除"提示

#### 1.5 Toast 交互
- [ ] 点击 Toast 右侧的 ✕ 按钮
- [ ] **预期结果**：Toast 立即关闭
- [ ] 按 ESC 键
- [ ] **预期结果**：Toast 立即关闭

#### 1.6 移动端显示
- [ ] 调整浏览器窗口到移动端尺寸（< 640px）
- [ ] 触发任何 Toast 通知
- [ ] **预期结果**：Toast 在屏幕顶部居中显示，左右有适当边距

### 2. 确认对话框

#### 2.1 删除任务确认
- [ ] 点击任务的删除按钮（✕）

#### 2.2 键盘快捷键

- [ ] **预期结果**：对话框关闭，任务未被删除
- [ ] 再次打开删除确认对话框
- [ ] 按 Enter 键
- [ ] **预期结果**：任务被删除

### 3. 任务优先级

#### 3.1 添加不同优先级的任务
- [ ] 打开任务列表
- [ ] 点击"高优先级"按钮（红色）
- [ ] 输入任务"紧急任务"，按 Enter
- [ ] **预期结果**：任务左侧显示红色圆点
- [ ] 点击"中优先级"按钮（黄色）
- [ ] 输入任务"普通任务"，按 Enter
- [ ] **预期结果**：任务左侧显示黄色圆点
- [ ] 点击"低优先级"按钮（绿色）
- [ ] 输入任务"不紧急任务"，按 Enter
- [ ] **预期结果**：任务左侧显示绿色圆点

#### 3.2 优先级视觉区分
- [ ] 观察任务列表
- [ ] **预期结果**：
  - 高优先级：红色圆点
  - 中优先级：黄色圆点
  - 低优先级：绿色圆点
  - 悬停在圆点上显示优先级文字
