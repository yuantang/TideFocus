# FlowMind 快速开始指南

## 🚀 启动应用

### 1. 安装依赖（首次运行）
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 打开浏览器
访问 `http://localhost:3000/`

## 🎵 音频功能使用指南

### 环境白噪音（背景音）

1. **选择音频**
   - 点击右下角设置图标 ⚙️
   - 在"音效设置"部分找到"Ambient Sound"
   - 从下拉菜单选择你喜欢的音频

2. **预览音频**
   - 选择音频后，点击右侧的播放按钮 ▶️
   - 试听音频效果
   - 再次点击可以停止播放 ⏸

3. **调节音量**
   - 在主界面底部找到音量滑块
   - 拖动滑块调节音量（0-100%）
   - 音量会平滑过渡，不会突兀

4. **开始使用**
   - 点击"保存"按钮
   - 点击播放按钮 ▶️ 开始计时
   - 环境音会自动循环播放

### 完成提示音

1. **启用提示音**
   - 打开设置 ⚙️
   - 找到"Completion Sound"
   - 勾选右侧的开关

2. **选择音效**
   - 从下拉菜单选择提示音
   - 点击播放按钮 ▶️ 试听
   - 推荐：颂钵（默认）- 温柔悠长

3. **触发时机**
   - 每个番茄钟结束时自动播放
   - 提醒你切换模式（专注→休息）

### 长休息提醒音

1. **启用提醒音**
   - 打开设置 ⚙️
   - 找到"Long Break Reminder"
   - 勾选右侧的开关

2. **选择音效**
   - 从下拉菜单选择提醒音
   - 点击播放按钮 ▶️ 试听
   - 推荐：摇铃（默认）- 温柔提醒

3. **触发时机**
   - 在长休息前的最后一个专注阶段开始时播放
   - 提前提醒你即将进入长休息

## 🎯 推荐音频组合

### 组合 1: 学习专注
- **环境音**: 雨声 (Rain) 或 图书馆 (Library)
- **完成提示音**: 颂钵 (Singing Bowl)
- **长休息提醒音**: 摇铃 (Bell)
- **适用场景**: 学习、阅读、考试准备

### 组合 2: 工作写作
- **环境音**: 咖啡店 (Cafe) 或 森林 (Forest)
- **完成提示音**: 竹风铃 (Bamboo Chime)
- **长休息提醒音**: 钢琴 (Piano)
- **适用场景**: 工作、写作、创作

### 组合 3: 深度专注
- **环境音**: 白噪音 (White Noise) 或 海洋 (Ocean)
- **完成提示音**: 音乐盒 (Music Box)
- **长休息提醒音**: 摇铃 (Bell)
- **适用场景**: 需要完全屏蔽外界干扰

### 组合 4: 放松冥想
- **环境音**: 风铃 (Wind Chimes) 或 壁炉 (Fireplace)
- **完成提示音**: 颂钵 (Singing Bowl)
- **长休息提醒音**: 钢琴 (Piano)
- **适用场景**: 冥想、放松、休息

## ⌨️ 快捷键

- **空格键**: 播放/暂停计时器
- **右箭头键**: 跳过当前阶段
- **Esc**: 退出全屏模式

## 🎨 主题切换

1. 打开设置 ⚙️
2. 找到"主题"部分
3. 选择预设主题：
   - **Rose (玫瑰)**: 温暖粉色调（默认）
   - **Forest (森林)**: 清新绿色调
   - **Sky (天空)**: 宁静蓝色调
   - **Dusk (黄昏)**: 深色模式

## 📊 查看统计

1. 点击右上角信息图标 ℹ️
2. 查看：
   - 今日完成的番茄钟数
   - 本周进度图表
   - 总计完成数
   - 连续专注天数

## 🔧 自定义设置

### 时间设置
- **专注时间**: 默认 25 分钟，可调整 1-60 分钟
- **短休息**: 默认 5 分钟，可调整 1-60 分钟
- **长休息**: 默认 15 分钟，可调整 1-60 分钟
- **每轮会话数**: 默认 4 次，可调整 0-12 次（0 表示禁用长休息）

### 目标设置
- **每日目标**: 默认 8 个番茄钟，可调整 0-24 个

### 颜色自定义
- 可以自定义专注、短休息、长休息的背景色和文字色
- 实时预览效果
- 一键重置为默认主题

## 💡 使用技巧

1. **首次使用**
   - 建议先试听所有音频，找到最适合自己的
   - 从默认的 25 分钟专注开始
   - 不要跳过休息时间

2. **音频选择**
   - 专注时选择不干扰思考的音频（雨声、白噪音、图书馆）
   - 休息时可以选择更放松的音频（森林、海洋、风铃）
   - 音量建议设置在 30-50%

3. **长期使用**
   - 设置合理的每日目标
   - 关注统计数据，了解自己的专注习惯
   - 定期更换音频，保持新鲜感

4. **全屏模式**
   - 点击全屏按钮进入沉浸式体验
   - 减少干扰，提升专注度
   - 按 Esc 退出全屏

## 🐛 常见问题

### 音频无法播放？
1. 检查浏览器是否允许自动播放
2. 尝试手动点击播放按钮
3. 检查音量是否为 0
4. 查看控制台是否有错误信息

### 音频加载慢？
1. 首次加载需要时间，请耐心等待
2. 音频会自动预加载，后续使用会更快
3. 确保网络连接正常（虽然音频在本地）

### 提示音没有播放？
1. 检查是否启用了提示音开关
2. 确认选择的不是"无声 (None)"
3. 检查系统音量是否静音
