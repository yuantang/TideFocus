# TideFocus 快速开始指南

## 🚀 启动应用

### 1. 安装依赖（首次运行）
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 打开浏览器
访问 `http://localhost:3000/`

## 🎵 白噪音混音器使用指南

### 基础使用

1. **打开混音器**
   - 点击主界面底部的"🎵 白噪音"按钮
   - 或点击右下角设置图标 ⚙️

2. **选择声音**
   - 浏览 5 个分类：白噪音、水声、氛围、自然、环境
   - 每个分类下有 4 种声音
   - 点击声音卡片即可播放/停止

3. **调节音量**
   - 每个声音都有独立的音量滑块
   - 拖动滑块调节音量（0-100%）
   - 可以同时播放多个声音

4. **使用预设**
   - 点击"预设"标签
   - 选择 8 个精选混音预设之一
   - 一键加载专业调配的声音组合

### 8 个精选预设

1. **🎯 深度专注** - 棕噪音 + 白噪音
2. **🌧️ 雨中学习** - 雨声 + 粉噪音 + 雷声
3. **🌊 海洋宁静** - 海洋 + 海浪 + 棕噪音
4. **🔥 温暖壁炉** - 壁炉 + 粉噪音 + 风声
5. **🌿 森林静修** - 森林 + 鸟鸣 + 白噪音
6. **☕ 咖啡馆工作** - 咖啡店 + 粉噪音
7. **📚 图书馆静谧** - 图书馆 + 棕噪音
8. **🌙 深夜工作** - 虫鸣 + 风铃 + 紫噪音

## ✅ 任务管理

### 创建任务

1. **打开任务列表**
   - 点击主界面的"📝 任务"按钮

2. **添加新任务**
   - 在输入框中输入任务内容
   - 按回车或点击"添加"按钮

3. **关联专注会话**
   - 开始专注前，在意图弹框中选择任务
   - 或直接输入新的专注内容

### 管理任务

- **完成任务**: 点击任务前的复选框
- **删除任务**: 点击任务右侧的删除按钮
- **查看历史**: 在任务列表中查看已完成的任务

## 🏆 成就系统

### 查看成就

1. **打开成就面板**
   - 点击右上角信息图标 ℹ️
   - 切换到"成就"标签

2. **成就分类**
   - 🎯 **专注成就**: 完成专注会话相关
   - 🔥 **连续成就**: 连续专注天数相关
   - ⏱️ **时长成就**: 累计专注时长相关
   - ✅ **任务成就**: 完成任务相关

3. **解锁成就**
   - 达到条件后自动解锁
   - 弹窗提示新成就
   - 查看进度条了解距离下一个成就还有多远

## 📊 数据统计

### 查看统计

1. **打开统计面板**
   - 点击右上角信息图标 ℹ️
   - 切换到"进度"标签

2. **查看数据**
   - 📅 **日视图**: 查看单日专注数据
   - 📊 **周视图**: 查看本周进度图表
   - 📈 **月视图**: 查看月度统计和趋势

3. **导出数据**
   - 在月视图中找到"数据导出"部分
   - 选择 JSON 或 CSV 格式
   - 下载数据到本地

## 🔧 自定义设置

### 时间设置
1. 打开设置 ⚙️
2. 在"计时器设置"部分调整：
   - **专注时间**: 默认 25 分钟（1-60 分钟）
   - **短休息**: 默认 5 分钟（1-60 分钟）
   - **长休息**: 默认 15 分钟（1-60 分钟）
   - **每轮会话数**: 默认 4 次（2-12 次）

### 目标设置
- **每日目标**: 默认 8 个番茄钟（1-24 个）
- 在主界面顶部查看今日进度

### 主题自定义
1. 在设置中找到"主题设置"
2. 选择预设主题或自定义颜色
3. 可以自定义：
   - 专注模式背景色和文字色
   - 短休息模式背景色和文字色
   - 长休息模式背景色和文字色
4. 实时预览效果
5. 一键重置为默认主题

## 💡 使用技巧

### 首次使用
- 建议先试听所有白噪音，找到最适合自己的
- 从默认的 25 分钟专注开始
- 不要跳过休息时间，休息同样重要

### 白噪音选择
- **专注时**: 选择不干扰思考的声音（棕噪音、白噪音、图书馆）
- **休息时**: 选择更放松的声音（森林、海洋、风铃）
- **音量建议**: 设置在 30-50%，不要过大
- **混音技巧**: 尝试组合 2-3 种声音，创造独特氛围

### 任务管理
- 在开始专注前明确任务目标
- 将大任务拆分成多个小任务
- 每完成一个番茄钟，标记任务进度
- 定期回顾已完成的任务，获得成就感

### 长期使用
- 设置合理的每日目标（建议 6-8 个番茄钟）
- 关注统计数据，了解自己的专注习惯
- 定期更换白噪音组合，保持新鲜感
- 追踪成就进度，保持动力

## 🐛 常见问题

### 白噪音无法播放？
1. 检查浏览器是否允许自动播放
2. 尝试手动点击声音卡片
3. 检查音量滑块是否为 0
4. 刷新页面重试

### 数据丢失了？
- 数据保存在浏览器 LocalStorage 中
- 清除浏览器数据会导致数据丢失
- 建议定期导出数据备份（月视图 → 数据导出）

### 如何切换语言？
1. 打开设置 ⚙️
2. 在"语言"部分选择语言
3. 支持 6 种语言：简体中文、繁体中文、English、Español、日本語、한국어
