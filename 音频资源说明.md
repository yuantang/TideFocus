## 🎵 音频资源详情

| 文件名 | 中文名 | 英文名 | 适用场景 | 特点 |
|--------|--------|--------|----------|------|
| rain.ogg | 雨声 | Rain | 专注学习、阅读 | 经典白噪音，屏蔽环境杂音 |
| forest.ogg | 森林 | Forest | 放松、冥想 | 自然鸟鸣，舒缓心情 |
| cafe.ogg | 咖啡店 | Cafe | 工作、写作 | 人声氛围，模拟咖啡馆环境 |
| fireplace.ogg | 壁炉 | Fireplace | 冬季学习、夜间工作 | 温暖舒适，噼啪声 |
| ocean.ogg | 海洋 | Ocean | 深度专注、放松 | 海浪声，平静规律 |
| wind-chimes.ogg | 风铃 | Wind Chimes | 轻度工作、休息 | 轻柔悦耳，不干扰思考 |
| white-noise.ogg | 白噪音 | White Noise | 需要完全屏蔽外界声音 | 纯白噪音，最强屏蔽效果 |
| library.ogg | 图书馆 | Library | 学习、考试准备 | 安静氛围，偶尔翻书声 |

| singing-bowl.ogg | 颂钵 | Singing Bowl | 温柔悠长，禅意十足 |

### 长休息提醒音（Reminder Sounds）

在长休息前的最后一个专注阶段开始时播放，提前提醒用户。

| 文件名 | 中文名 | 英文名 | 特点 |
|--------|--------|--------|------|

## 🎨 音频选择理念

### 1. 温柔舒缓
所有音频都经过精心挑选，确保：
- 不刺耳、不突兀
- 音量适中，可调节
- 适合长时间聆听

### 2. 自然真实
- 优先选择自然环境音（雨声、森林、海洋）
- 避免过度合成的电子音
- 保留音频的自然质感

### 3. 文化融合
- 中英文双语命名
- 融合东方元素（颂钵、竹风铃）
- 兼顾西方经典（音乐盒、钢琴）

### 4. 场景适配
- 不同场景提供不同选择
- 学习、工作、放松各有侧重
- 支持完全静音模式

## 🔧 技术规格

### 音频格式
- **格式**: OGG Vorbis
- **优势**: 
  - 开源免费
  - 压缩率高，文件小
  - 音质优秀
  - 浏览器兼容性好

### 文件大小
- 环境音: 约 100KB - 1MB（循环片段）
- 提示音: 约 100KB - 800KB（短音效）
- 提醒音: 约 100KB - 300KB（短音效）

### 播放特性
- **环境音**: 
  - 循环播放（loop: true）
  - 淡入淡出效果（1秒）
  - 音量可调（0-100%）
- **提示音/提醒音**: 
  - 单次播放
  - 固定音量
  - 可开关

## 📝 使用说明

### 在代码中引用

音频路径在 `constants.ts` 中定义：

```typescript
export const SOUNDS: Sound[] = [
  { id: 'rain', name: '雨声 (Rain)', url: '/sounds/ambient/rain.ogg' },
  // ...
];
```

### 添加新音频

1. 将音频文件放入对应目录：
   - 环境音 → `public/sounds/ambient/`
   - 完成提示音 → `public/sounds/completion/`
   - 提醒音 → `public/sounds/reminder/`

2. 更新 `constants.ts`：
   ```typescript
   { id: 'new_sound', name: '新音频 (New Sound)', url: '/sounds/ambient/new_sound.ogg' }
   ```

3. 确保文件格式为 `.ogg`
